Entity Resolution Service (ERS) v1.0.0
Service Specification

Summary

The ERS determines a match score for each pair-wise record attribute provided to the service. The ERS creates a merged entity record based on a pair-wise match threshold analysis of record attributes provided to the service.  

Description

The Entity Resolution Service (ERS) resolves and correlates entities across multiple data sources.  Law enforcement support systems (e.g., federated query or link analysis applications) will use this service whenever they need to determine (at some probability) whether two entities are the same based on similair attributes within a record.
Note: The ERS service allows the user to specify which algorithm to use for the match/merge functions.
