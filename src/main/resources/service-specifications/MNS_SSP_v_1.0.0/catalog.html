Merge Notification Service (MNS) v1.0.0
Service Specification

Summary

This service will provide information about individual system records that, based on certain criteria, represent the same real-world entity.     

Description

This service is able to provide information to interested parties about individual system records that probably represent the same real world.  This type of information can facilitate investigations by providing a sense of probability that otherwise independent pieces of information are actually about the same entity. 

Law enforcement support systems (e.g., federated query or link analysis applications) will use this service whenever they need to determine (at some probability) whether two entities are the same based on common attributes within a record.
